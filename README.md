### Init project nodejs
```bash
npm init
npm init -y
```

### Install dep
```
npm i <name> // prod dependencies
npm i -D <name> // dev dependencies 
```

- GET /event - get all events
- GET /event/:id - get event by id
- POST /event - create event
- DELETE /event/:id - delete event - by id
- PUT /event/:id - update event by id
